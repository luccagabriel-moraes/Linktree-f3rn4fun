# 👁️ f3rn4fun - Linktree Project
Este é um agregador de links personalizado (estilo Linktree) desenvolvido como parte dos meus estudos de desenvolvimento web. O projeto apresenta um design minimalista com foco em UI/UX, incluindo efeitos de vidro e alternância entre temas.

## 🚀 Funcionalidades
Agregador de Links: Centraliza redes sociais como Instagram, Twitter e WhatsApp em um só lugar.

Toggle de Tema (Dark/Light): Alternância dinâmica entre fundo claro e escuro, mantendo a identidade visual do logotipo central.

Design Glassmorphism: Botões com efeito de transparência e desfoque de fundo (backdrop-filter).

Efeito de Vinheta: Bordas com sombreamento suave para focar a atenção no conteúdo central.
